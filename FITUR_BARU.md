## 🎨 Customization Tips

### Mengubah Warna
Edit di `styles.css` `:root` section:
```css
--primary-color: #d4af37;      /* Emas */
--secondary-color: #8b7355;    /* Cokelat */
--accent-color: #e91e63;       /* Pink */
```

### Mengubah Font
Update di `styles.css`:
```css
body {
    font-family: 'Your Font', sans-serif;
}
```

### Mengubah Animasi Speed
Ubah `--transition` variable:
```css
--transition: all 0.3s cubic-bezier(...);
```
