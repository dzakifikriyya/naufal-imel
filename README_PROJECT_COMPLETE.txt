❓ FAQ
═══════════════════════════════════════════════════════════

Q: Apakah data RSVP aman?
A: Data disimpan di localStorage browser (local only, tidak ke cloud)
   Backup dengan export berkala

Q: Bisakah 2 orang akses admin bersamaan?
A: Bisa, tapi data di browser masing-masing
   Recommended: 1 orang yang manage admin

Q: Apakah bisa akses dari luar WiFi yang sama?
A: Tidak (IP local hanya dalam 1 network)
   Untuk luar, perlu hosting online (berbayar)

Q: Data hilang jika cache dihapus?
A: Ya, localStorage akan hilang
   Oleh karena itu, backup berkala sangat penting!

Q: Berapa maksimal tamu yang bisa handle?
A: Unlimited (tergantung browser memory)
   Tested sampai 1000+ RSVP masih lancar

Q: Bisa export data ke Excel?
A: Ya, buka admin → tab Export → download Excel

Q: Bagaimana kalau lupa password admin?
A: Edit file admin.html, ganti const ADMIN_PASSWORD

🚨 IMPORTANT NOTES
═══════════════════════════════════════════════════════════

1. DATA LOCAL ONLY:
   Data hanya tersimpan di browser local (localStorage)
   Bukan cloud storage
   Backup dengan export SANGAT PENTING!

2. SINGLE DEVICE:
   Setiap device (HP/komputer) punya storage terpisah
   Data tidak sinkron antar device
   Recommended: Monitor dari 1 komputer utama

3. NETWORK LIMITED:
   Hanya accessible di WiFi yang sama
   Untuk akses dari luar, butuh setup lanjutan

4. SESSION TIMEOUT:
   Admin dashboard timeout setelah refresh
   Password hanya simpan di session (aman)

5. BACKUP CRITICAL:
   Export data setiap hari
   Jangan andalkan local storage saja
