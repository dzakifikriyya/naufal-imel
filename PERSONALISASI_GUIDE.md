✨ FITUR BARU YANG DITAMBAHKAN:

1. 🔗 URL Parameter (?to=nama)
   - Menampilkan nama tamu di halaman undangan
   - Nama langsung di-encode di URL
   - Contoh: http://192.168.1.100/wedding/?to=Ipal%20%26%20Partner

2. 🌐 URL Generator (url-generator.html)
   - Tool untuk membuat link personal
   - Bisa share via WhatsApp langsung
   - Bisa share via Email

3. 📱 Hub Page (hub.html)
   - Akses semua fitur dari satu halaman
   - Navigasi yang mudah

🔍 DEBUGGING - Jika Ada Masalah

Masalah: Nama tidak tampil
─────────────────────────
✓ Pastikan URL format: ?to=NamaTamu
✓ Nama harus di-encode (gunakan URL Generator)
✓ Refresh halaman (Ctrl+F5)
✓ Buka F12 → Console → cek error

Masalah: URL tidak copy
──────────────────────
✓ Gunakan browser modern (Chrome, Firefox, Safari)
✓ Cek permission clipboard
✓ Copy manual dari text box

Masalah: WhatsApp tidak terbuka
───────────────────────────────
✓ WhatsApp Web sudah login
✓ Atau gunakan Mobile browser (lebih reliable)
