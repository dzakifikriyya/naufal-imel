📱 Setup Akses di Handphone

Panduan lengkap membuka website undangan di mobile device

⚡ Quick Start (5 Menit)

🎯 Rekomendasi Tercepat: Gunakan metode IP Address untuk WiFi rumah atau ngrok untuk akses dari mana saja.
1

Pastikan Laragon Berjalan

✅ Laragon harus aktif dengan Apache hijau

✅ Website di folder: c:\laragon\www\wedding

2

Cari IP Address Komputer

Buka Command Prompt dan ketik:

ipconfig

Catat IPv4 Address (contoh: 192.168.1.100)

3

Buka di Handphone

Browser handphone → ketik:

http://192.168.1.100/wedding

(Ganti IP sesuai hasil langkah 2)

4

Selesai! 🎉

Website akan muncul di handphone Anda

🔧 Pilih Metode Sesuai Kebutuhan

Opsi 1: IP Address (Recommended untuk WiFi)

Cocok untuk: Akses dari WiFi rumah/kantor yang sama

Kelebihan: Mudah setup, cepat, tidak perlu tools tambahan

Kekurangan: Hanya dalam WiFi sama, IP bisa berubah

MUDAH

Opsi 2: ngrok (Akses Global)

Cocok untuk: Akses dari mana saja (luar WiFi)

Kelebihan: Bisa diakses dari internet publik, cocok untuk testing

Kekurangan: Perlu setup tool, URL berubah setiap restart

SEDANG

Opsi 3: Hosting Online (Professional)

Cocok untuk: Acara formal, perlu permanent link

Kelebihan: Stabil, professional, shareable link permanen

Kekurangan: Butuh setup, bisa berbayar

KOMPLEKS

✅ Pre-Launch Checklist

📲 Bagikan via QR Code

💡 Pro Tip: Generate QR Code dari URL untuk sharing yang lebih mudah!

Steps:

  1. Kunjungi qr-code-generator.com
  2. Input URL website (IP Address atau ngrok)
  3. Download QR Code
  4. Bagikan ke undangan (WhatsApp, Email, Cetak)

🔍 Troubleshooting

❌ "ERR_CONNECTION_REFUSED"
Laragon tidak berjalan. Solusi: Buka Laragon > Start All
❌ "ERR_NAME_NOT_RESOLVED"
IP Address salah atau HP tidak di WiFi sama. Solusi: Cek ipconfig ulang, pastikan WiFi sama
❌ "Tampilan berantakan di HP"
Cache lama. Solusi: Clear cache browser atau buka Incognito mode
